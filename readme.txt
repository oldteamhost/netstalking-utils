NETSTALKING-PACK

1. imghst - для парсинга картинок с фотохостингов, хорошо работает для imgur и lightshot,
подерживает и конфиги, можно настроить под почти любой сайт, (в комплект идут конфиги).

2. icmpflood - простой icmp дудос, положить сайт врятли выйдет, но какой нибудь игровой
сервер думаю можно нагрузить хорошо.

3. lasth - Поиск информации, оставленной человеком в Интернете. Вы задаете программе имя,
и она ищет страницы в социальных сетях и веб-сайты с этим именем.

Для компиляции установить libcurl, и запустить "Makefile" командой make.
